# MiMo Token Plan Test Plan

## 目标

本测试计划用于评估 MiMo 模型在 AI Coding Agent 场景下的实际表现，重点关注代码理解、多文件修改、测试生成、Bug 修复、中文需求理解和长上下文任务能力。

## 测试工具

- Claude Code
- Codex
- OpenCode
- Cline
- Aider
- MiMo Token Plan

## 测试任务

1. 阅读已有项目代码并总结模块结构
2. 根据中文需求生成实现方案
3. 修改多个文件完成一个功能
4. 自动生成单元测试
5. 根据报错信息定位 Bug
6. 根据 Git diff 生成 PR 描述
7. 对比不同模型输出质量

## 记录维度

- 是否正确理解需求
- 是否能修改正确文件
- 是否产生幻觉
- 是否能通过测试
- 代码质量是否稳定
- 多轮对话后是否保持上下文
- 中文技术需求理解是否准确

## 后续计划

后续会持续记录 MiMo、Claude、GPT、DeepSeek 等模型在相同任务下的输出结果，并整理成功案例、失败案例和 bad case，用于模型能力反馈。
